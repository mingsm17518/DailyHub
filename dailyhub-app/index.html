使用说明

  1. 访问 EmailJS 并注册账号
  2. 添加邮件服务（Gmail、Outlook 或自定义 SMTP）
  3. 创建邮件模板，使用以下变量：
    • {{to_email}} - 接收邮箱
    • {{to_name}} - 接收人名称
    • {{event_title}} - 日程标题
    • {{event_date}} - 日程日期
    • {{event_time}} - 日程时间
    • {{reminder_type}} - 提醒类型
    • {{description}} - 备注
  4. 填写上述 Service ID、Template ID 和 Public Key
  5. 点击"发送测试邮件"验证配置
